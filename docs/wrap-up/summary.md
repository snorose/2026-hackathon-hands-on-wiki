# 오늘 만든 AI 에이전트 정리

이번 핸즈온에서는 작은 AI 에이전트 흐름을 만들었습니다.

## 전체 흐름

1. 사용자가 노트를 입력합니다.
2. n8n Webhook이 요청을 받습니다.
3. 파일이면 OCR로 글자를 읽습니다.
4. 참고자료를 함께 넣어 프롬프트를 만듭니다.
5. AI가 피드백을 만듭니다.
6. AI가 OX 퀴즈를 만듭니다.
7. 결과가 프론트 화면에 표시됩니다.

## 배운 개념

- Trigger
- Tool use
- OCR
- Pseudo-RAG
- Prompt
- Structured output
- Memory

## 중요한 포인트

AI 에이전트는 한 번에 마법처럼 답하는 것이 아닙니다.

입력을 받고, 필요한 도구를 호출하고, 중간 결과를 정리하고, 다시 다음 단계로 넘기는 흐름으로 구성됩니다.
