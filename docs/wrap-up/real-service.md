# 실제 서비스와 핸즈온 버전의 차이

이번 핸즈온 버전은 학습을 위해 단순화되어 있습니다.

## 제외한 기능

- 로그인
- 회원가입
- 피드백 보관함
- 학습 데이터 관리
- 사지선다 퀴즈
- 별도 서비스 DB
- 벡터DB

## 왜 제외했나요?

처음부터 모든 기능을 넣으면 핵심 흐름을 이해하기 어렵습니다.

이번 실습의 목표는 AI 서비스의 핵심 구조를 보는 것입니다.

```text
입력 -> 도구 호출 -> 참고자료 반영 -> AI 생성 -> 결과 표시
```

## 실제 서비스에서는 무엇이 추가될까요?

실제 서비스에서는 사용자의 데이터를 저장하고, 다시 볼 수 있어야 합니다.

또한 여러 참고자료 중 관련 있는 내용을 자동으로 찾아야 합니다.

그래서 다음 요소가 추가될 수 있습니다.

- 사용자 DB
- 노트 DB
- 피드백 DB
- 벡터DB
- 인증 시스템
- 관리자 화면

이번 실습을 이해했다면, 그 위에 하나씩 기능을 추가할 수 있습니다.
