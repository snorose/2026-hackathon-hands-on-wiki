# 더 발전시키면 할 수 있는 것

이번 실습은 핵심 흐름만 남긴 버전입니다.

여기서 더 발전시키면 실제 서비스에 가까워집니다.

## 확장 아이디어

| 확장 | 설명 |
| --- | --- |
| 참고자료 PDF 자동 OCR | 참고자료도 PDF로 업로드하고 자동으로 텍스트화합니다. |
| 실제 RAG | 여러 문서를 저장하고 관련 부분을 검색합니다. |
| 피드백 보관함 | AI 피드백을 저장해 나중에 다시 봅니다. |
| 학습 이력 관리 | 어떤 노트를 언제 분석했는지 기록합니다. |
| 사지선다 퀴즈 | OX보다 더 다양한 문제를 만듭니다. |
| 로그인 | 사용자별로 데이터를 분리합니다. |

## 추천 확장 순서

1. 참고자료 PDF 자동 OCR
2. 피드백 저장
3. 실제 RAG
4. 로그인과 학습 이력
