# 왜 별도 프론트를 쓰나요?

비전공자 핸즈온에서는 n8n 안에서 화면까지 만들기보다, 완성된 프론트 화면을 따로 실행하는 편이 더 쉽습니다.

## 이유

1. 사용자는 익숙한 웹 화면에서 입력합니다.
2. 파일 업로드와 결과 표시가 자연스럽습니다.
3. n8n에서는 자동화 흐름에만 집중할 수 있습니다.

## n8n에서 보는 것

n8n에서는 아래 흐름을 봅니다.

- 요청 받기
- 입력 정리하기
- OCR 호출하기
- 프롬프트 만들기
- AI 호출하기
- 결과 정리하기

## 프론트에서 보는 것

프론트에서는 아래 경험을 봅니다.

- 노트 입력
- 이미지/PDF 업로드
- 참고자료 입력
- AI 피드백 확인
- OX 퀴즈 풀기
