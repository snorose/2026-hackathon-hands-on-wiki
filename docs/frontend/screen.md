# 화면 구성

프론트 화면은 실습에 필요한 최소 기능만 담고 있습니다.

## 입력 영역

| 항목 | 설명 |
| --- | --- |
| n8n Webhook URL | 프론트가 요청을 보낼 n8n 주소입니다. |
| 참고자료 | AI가 참고할 자료를 붙여넣는 곳입니다. |
| 텍스트 | 노트를 직접 입력하는 모드입니다. |
| 이미지/PDF | 노트 파일을 업로드하는 모드입니다. |
| 분석하기 | n8n 워크플로우를 실행하는 버튼입니다. |

## 결과 영역

| 항목 | 설명 |
| --- | --- |
| AI 피드백 | 노트의 잘못된 개념이나 칭찬을 보여줍니다. |
| OX 퀴즈 | 노트를 바탕으로 만든 OX 문제를 보여줍니다. |
| 정답 확인 | O 또는 X를 누르면 정답과 설명이 표시됩니다. |

## 왜 제목과 과목이 없나요?

이번 실습은 AI 에이전트 흐름을 이해하는 것이 목표입니다.

제목과 과목은 저장 기능이나 학습 관리 기능이 있을 때 유용합니다. 하지만 이번 버전에서는 결과를 저장하지 않으므로 제거했습니다.
