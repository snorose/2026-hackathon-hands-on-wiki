# 필요한 것 한눈에 보기

아무것도 설치되어 있지 않은 컴퓨터라고 가정하고 시작합니다.

## 필요한 도구

| 필요한 것 | 왜 필요한가요? |
| --- | --- |
| 웹 브라우저 | n8n과 실습 화면을 열기 위해 필요합니다. |
| 터미널 | 명령어를 실행하는 창입니다. |
| Docker Desktop | n8n을 쉽게 실행하기 위해 필요합니다. |
| 텍스트 편집기 | `.env` 파일을 수정할 때 사용합니다. |
| Upstage API Key | OCR과 AI 답변 생성을 사용하기 위한 열쇠입니다. |
| 실습 폴더 | 프론트 화면과 n8n 워크플로우 파일이 들어 있습니다. |

## n8n을 따로 설치하지 않습니다

n8n은 직접 설치하지 않습니다. Docker Desktop을 설치한 뒤, 명령어 하나로 n8n을 실행합니다.

```bash
docker compose up -d
```

이 명령어는 “n8n을 담은 실행 상자를 내려받고, 백그라운드에서 켜 주세요”라는 뜻입니다.

## 전체 준비 순서

1. 터미널을 엽니다.
2. Docker Desktop을 설치하고 실행합니다.
3. Upstage API Key를 준비합니다.
4. 실습 폴더를 준비합니다.
5. `.env` 파일에 API Key를 넣습니다.
6. n8n과 프론트 화면을 실행합니다.
