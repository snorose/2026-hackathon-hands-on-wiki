# n8n은 왜 Docker로 실행하나요?

n8n은 자동화 워크플로우를 만드는 도구입니다. 직접 설치할 수도 있지만, 처음 배우는 사람에게는 설치 과정이 복잡할 수 있습니다.

그래서 이번 실습에서는 Docker로 n8n을 실행합니다.

## 직접 설치와 Docker 실행의 차이

| 방식 | 설명 |
| --- | --- |
| 직접 설치 | 내 컴퓨터에 n8n 실행 환경을 하나씩 설치합니다. |
| Docker 실행 | 이미 준비된 n8n 상자를 받아서 실행합니다. |

이번 실습에서는 두 번째 방식을 사용합니다.

```bash
docker compose up -d
```

이 명령어 하나로 n8n을 실행할 수 있습니다.

## Docker Compose란?

Docker Compose는 여러 실행 설정을 파일 하나에 적어 두고 한 번에 실행하는 도구입니다.

실습 폴더에는 `docker-compose.yml` 파일이 있습니다. 이 파일 안에 “n8n을 어떤 포트로 실행할지”, “데이터를 어디에 저장할지”가 적혀 있습니다.
