# 터미널 아주 가볍게 이해하기

터미널은 컴퓨터에게 글자로 명령을 내리는 창입니다. 이번 실습에서는 복잡한 명령어를 외우지 않아도 됩니다. 문서에 있는 명령어를 복사해서 붙여넣으면 됩니다.

## macOS에서 터미널 열기

1. `Command + Space`를 누릅니다.
2. `Terminal` 또는 `터미널`을 검색합니다.
3. 실행합니다.

## Windows에서 터미널 열기

1. 시작 메뉴를 엽니다.
2. `PowerShell` 또는 `Terminal`을 검색합니다.
3. 실행합니다.

## 폴더 이동하기

터미널에서 특정 폴더로 이동할 때는 `cd` 명령어를 씁니다.

```bash
cd "2026-hackathon-hands-on"
```

`cd`는 “change directory”, 즉 “폴더를 바꿔 주세요”라는 뜻입니다.

## 명령어를 복사해서 붙여넣기

문서의 코드 블록은 그대로 복사해서 터미널에 붙여넣을 수 있습니다.

```bash
docker compose up -d
```

명령어를 입력한 뒤에는 `Enter`를 누릅니다.
