# 참고자료 텍스트 넣어보기

참고자료 칸은 AI가 피드백과 퀴즈를 만들 때 기준으로 삼을 자료를 넣는 곳입니다.

## 실습 방법

1. 프론트 화면을 엽니다.
2. 참고자료 칸에 교과서나 수업자료 일부를 붙여넣습니다.
3. 노트를 입력합니다.
4. `분석하기`를 누릅니다.

## 예시 참고자료

```text
소프트웨어 공학은 소프트웨어를 체계적으로 개발하고 유지보수하기 위한 기술, 방법, 절차를 다루는 분야이다.
```

## 예시 노트

```text
소프트웨어 공학은 코딩만 빠르게 하는 방법이다.
```

## 기대되는 피드백

AI는 참고자료를 보고 “소프트웨어 공학은 코딩만을 의미하지 않고, 개발과 유지보수를 체계적으로 다루는 분야”라고 피드백할 수 있습니다.

## 비교해 보기

같은 노트를 아래 두 가지 방식으로 실행해 보세요.

1. 참고자료 없이 실행
2. 참고자료를 넣고 실행

참고자료가 있을 때 피드백이 더 수업자료에 가까워지는지 확인합니다.
