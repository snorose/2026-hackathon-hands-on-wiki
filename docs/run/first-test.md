# 첫 번째 테스트하기

이제 실제로 노트를 넣고 AI 피드백과 OX 퀴즈를 받아 봅니다.

## 텍스트 노트 테스트

1. 프론트 화면을 엽니다.
2. 입력 방식을 `텍스트`로 둡니다.
3. 노트 내용에 아래 문장을 넣습니다.

```text
광합성은 식물이 빛 에너지를 이용해 이산화탄소와 물로부터 포도당을 만드는 과정이다.
```

4. `분석하기` 버튼을 누릅니다.
5. AI 피드백과 OX 퀴즈가 나오는지 확인합니다.

## 참고자료를 넣고 다시 테스트

참고자료 칸에 아래 문장을 넣습니다.

```text
광합성은 엽록체에서 일어나며, 빛 에너지를 화학 에너지로 전환하는 과정이다.
```

같은 노트를 다시 분석해 봅니다.

참고자료가 있을 때 AI가 더 구체적으로 판단할 수 있습니다.

## 이미지/PDF 노트 테스트

1. 입력 방식을 `이미지/PDF`로 바꿉니다.
2. 글자가 들어 있는 이미지나 PDF를 선택합니다.
3. `분석하기` 버튼을 누릅니다.
4. OCR 후 피드백과 퀴즈가 나오는지 확인합니다.

## 결과가 나오면 성공

다음 중 하나라도 확인되면 흐름이 정상적으로 동작한 것입니다.

- AI 피드백이 표시된다.
- OX 퀴즈가 생성된다.
- 퀴즈에서 O 또는 X를 누르면 정답 설명이 표시된다.
