# OCR 도구 호출하기

OCR은 이미지나 PDF 안에 있는 글자를 읽어 텍스트로 바꾸는 기술입니다.

이번 실습에서는 Upstage OCR API를 사용합니다.

## 언제 OCR을 사용하나요?

사용자가 텍스트를 직접 입력하면 OCR이 필요 없습니다.

하지만 이미지나 PDF를 업로드하면 AI가 바로 읽을 수 없기 때문에 먼저 OCR을 거칩니다.

```text
이미지/PDF -> OCR -> 텍스트
```

## n8n에서의 흐름

1. `Has Document?` 노드가 파일이 있는지 확인합니다.
2. 파일이 있으면 `Upstage OCR` 노드로 보냅니다.
3. OCR 결과의 `text` 값을 꺼냅니다.
4. 그 텍스트를 피드백/퀴즈 생성에 사용합니다.

## OCR이 실패할 수 있는 경우

- 이미지가 너무 흐린 경우
- 글자가 너무 작은 경우
- 손글씨가 복잡한 경우
- PDF가 너무 큰 경우

이럴 때는 더 선명한 파일을 사용하거나 텍스트로 직접 입력해 볼 수 있습니다.
