# 프롬프트 만들기

프롬프트는 AI에게 보내는 요청문입니다.

“이 노트를 보고 피드백해 줘”라고만 쓰면 답변 형식이 매번 달라질 수 있습니다.

그래서 n8n은 AI에게 더 구체적으로 요청합니다.

## 피드백 프롬프트의 구성

피드백 프롬프트에는 다음 정보가 들어갑니다.

- AI의 역할: 선생님
- 판단 기준: 참고자료가 있으면 우선 참고
- 출력 형식: XML
- 참고자료: `<정보>` 영역
- 사용자의 노트: `<입력>` 영역

## 참고자료 영역

```xml
<정보>
참고자료 내용
</정보>
```

이 영역은 AI가 피드백할 때 기준으로 삼는 자료입니다.

## 입력 영역

```xml
<입력>
사용자가 작성한 노트
</입력>
```

이 영역은 실제로 검사할 노트입니다.

## 왜 형식을 정하나요?

프론트 화면은 AI 답변을 읽어서 예쁘게 보여줘야 합니다.

답변 형식이 매번 다르면 화면에서 처리하기 어렵습니다. 그래서 XML, JSON처럼 정해진 형식을 요청합니다.
