# 이미지/PDF 업로드 오류가 날 때

이미지/PDF 업로드는 OCR을 거칩니다.

OCR은 파일 속 글자를 읽어 텍스트로 바꾸는 과정입니다.

## 파일을 선택했는지 확인

이미지/PDF 모드를 선택한 뒤 파일을 선택해야 합니다.

파일을 선택하지 않으면 아래와 같은 메시지가 나올 수 있습니다.

```text
이미지 또는 PDF 파일을 선택해주세요.
```

## OCR이 글자를 못 읽는 경우

다음 파일은 OCR 결과가 좋지 않을 수 있습니다.

- 글자가 흐린 이미지
- 너무 작은 글자
- 회전된 이미지
- 복잡한 배경이 있는 이미지
- 손글씨가 많은 이미지

## 파일이 너무 큰 경우

큰 PDF는 업로드와 OCR에 시간이 오래 걸릴 수 있습니다.

처음 테스트할 때는 1-2페이지 정도의 작은 파일을 추천합니다.

## 워크플로우를 다시 import해야 하는 경우

워크플로우 JSON을 수정했다면 n8n에 다시 import해야 합니다.

기존에 import한 워크플로우는 파일 수정 내용을 자동으로 알지 못합니다.

## 업로드가 안 될 때

처음 테스트할 때는 1-2페이지 정도의 작은 파일을 추천합니다.

업로드했는데 텍스트가 없다고 나오면 최신 워크플로우 JSON을 다시 import하고 프론트 페이지를 새로고침하세요.
