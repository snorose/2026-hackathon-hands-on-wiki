# AI 응답 형식이 이상할 때

AI는 가끔 요청한 형식을 완벽하게 지키지 않을 수 있습니다.

이번 실습에서는 피드백은 XML, 퀴즈는 JSON으로 요청합니다.

## 가능한 증상

- 피드백이 카드 형태로 나오지 않고 긴 글로 나온다.
- OX 퀴즈가 비어 있다.
- 정답 버튼을 눌러도 설명이 이상하다.

## 왜 이런 일이 생기나요?

AI는 문장을 생성하는 모델입니다. 프롬프트로 형식을 강하게 요청해도 가끔 다른 형식으로 답할 수 있습니다.

## 해결 방법

1. 다시 실행해 봅니다.
2. 노트 내용을 조금 더 짧고 명확하게 입력합니다.
3. 참고자료를 너무 길게 넣지 않습니다.
4. n8n의 `Prepare Prompts` 노드에서 출력 형식 요청을 더 강하게 만듭니다.

## 실습에서 배울 점

AI 서비스에서는 답변 내용뿐 아니라 답변 형식도 중요합니다.

그래서 실제 서비스에서는 구조화된 출력, 검증, 재시도 같은 장치를 추가합니다.

# 출력 형식이 깨질 때

AI 응답이 기대한 JSON 또는 XML 형태와 다르면 프롬프트에서 출력 형식 요청을 더 강하게 만들어야 합니다.

n8n의 `Prepare Prompts` 노드에서 출력 예시와 금지 사항을 확인하세요.
